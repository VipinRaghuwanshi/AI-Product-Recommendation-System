# 🛍️ AI Product Recommendation System

An AI-powered product recommendation web application built with React and Groq AI API. Users can enter their preferences in natural language, and the AI recommends the most suitable products from the available catalog.

---

## 🚀 Features

- 🤖 AI-powered product recommendations
- 🔍 Natural language product search
- 📱 Product filtering based on user preferences
- ⚡ Fast response using Groq LLM API
- 🎨 Modern and responsive user interface
- 📋 Dynamic product catalog display

---

## 📸 Example Queries

Try searching with:

- Phone under $500
- Cheapest phone
- Best laptop under $700
- Budget smartphone
- Student laptop

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- JavaScript

### AI Integration
- Groq API
- Llama 3.3 70B Versatile Model

### Styling
- CSS3
- Responsive Design

---

## ⚙️ How It Works

1. User enters a product preference.
2. The preference is sent to the Groq AI model.
3. AI analyzes the available product catalog.
4. AI returns matching product recommendations.
5. The frontend filters and displays the recommended products.

## 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/VipinRaghuwanshi/AI-Product-Recommendation-System.git
```

Move to the project folder:

```bash
cd AI-Product-Recommendation-System
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_GROQ_API_KEY=YOUR_GROQ_API_KEY
```

Start the development server:

```bash
npm run dev
```
